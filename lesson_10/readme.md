# Инструкция по получению результатов выполенения домашнего задания к уроку 10
## Выполните следующие команды, находясь в той же директории, где файлы домашнего задания:
### pytest --alluredir=./allure-result
### allure serve allure-result/

## Также можно использовать команду для формирования отчета в отдельном html файле:
### allure generate allure-result/

## Чтобы открыть ранее сгенерированный в файле отчет используется команда:
### allure open allure-report/

## В результате получается сформированный отчет, который можно изучить
