# Инструкция по получению результатов выполенения домашнего задания к уроку 10
## Выполните следующие команды, находясь в той же директории, где файлы домашнего задания:
### pytest --alluredir=./allure-result
### allure serve allure-result/
## Изучите отчет выполнения теста, который открылся в браузере
